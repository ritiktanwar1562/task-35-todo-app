# 📝 Todo App (MERN Stack Project)

I made this Todo App while learning full stack development using MERN stack (MongoDB, Express, React, Node.js).

This project helped me understand how frontend and backend actually connect and work together using APIs.

---

# 🚀 What I Built

In this project, I built a simple Todo application where:

- I can add new tasks
- I can see all tasks
- I can edit tasks
- I can delete tasks
- Data is stored in MongoDB database

---

# 🧠 What I Learned

While making this project, I learned:

- How frontend (React) talks to backend using API calls
- How REST APIs work (GET, POST, PUT, DELETE)
- How MongoDB stores data in collections
- How Node.js and Express handle server-side logic
- How to structure a full stack project properly
- How CRUD operations work in real projects

---

# ⚙️ How I Built It

First, I created the backend using Node.js and Express.

Then I connected it with MongoDB to store data.

After that, I created frontend using React and connected it with backend using fetch API.

Every action like add, delete, edit sends request to backend and updates database.
